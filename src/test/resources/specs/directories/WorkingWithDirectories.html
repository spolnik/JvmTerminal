JvmTerminal - working with directories!

After a user runs terminal, he creates new directory and display content of working directory to see newly created directory. Then he goes into it and displays working directory path. Later he goes back to parent directory, he removes newly created directory and finally list content to see if his directory was removed.

Example

When user opens terminal he creates new directory dir_name, then he displays content to confirm it was created.

Then he goes to newly created directory and displays working directory path which should ends with dir_name.

Later he goes back to parent, he removes newly created directory and finally he lists content to see if the directory was removed.
