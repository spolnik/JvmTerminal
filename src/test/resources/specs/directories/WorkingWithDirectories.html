JvmTerminal - working with directories!

After a user runs terminal, he creates new directory and display content of working directory to see a new directory. Then he goes into it and displays working directory path. Later he goes back to parent directory, he removes created directory and finally he lists content to see if a directory was removed.

Creating and removing new directory

When user opens terminal he creates new directory dir_name, then he lists content of working directory to confirm it was created.

Then he goes to newly created directory and displays working directory path which should ends with dir_name.

Later he goes back to parent, he removes created directory and finally he lists content to see if the directory was removed.
