JvmTerminal - working with directories!

After a user runs terminal, he creates new directory and display content of working directory to see a new directory. Then he goes into it and displays working directory path. Later he goes back to parent directory, he removes created directory and finally he lists content to see if a directory was removed.

Creating and removing new directory

When user opens terminal he runs command mkdir dir_name to create a new directory, then using ls command he lists content of working directory to confirm dir_name directory was created.

After that he runs cd dir_name command to go to newly created directory and then he displays working directory path using pwd command which result of should end with dir_name.

Later he runs cd .. command to go back to parent, he removes created directory by using rmdir dir_name command and finally using ls he lists content to see if the directory was removed.
